です

Present Tense

Xは Y です

X is Y.

Examples:

学生です。
I am a student.

十二時半です。
It is half past 12.

山下さんは先生です。
Mr. Yamashita is a teacher.

Xは Y じゃないです

X is not Y.

Examples:

学生じゃないです。
I am a student.

十二時半じゃないです。
It is half past 12.

山下さんは先生じゃないです。
Mr. Yamashita is a teacher.

Past Tense

Xは　Y　でした。
X was Y.

Xは　Y　じゃなかったです。
X was not Y.

Examples:

学生じゃないでした。
I was a student.

十二時半でした。
It was half past 12.

山下さんは先生じゃなかったです。
Mr. Yamashita was not a teacher.

ある・いる

Present Tense

Non-living things:

X が　あります。
X is there.

X が　ありません。
X is not there.

Examples:

あそこにマクドナルドがあります。
There is a McDonald's over there.

火曜日に試験があります。
There is a test on Tuesday.

時間がありません。
I do not have time.

Living, moving things:

X　が　います。
X is there.

X が　いません。
X is not there.

Examples:

家で猫がいます。
There is a cat in the house.

あそこに留学生がいます。
There is a foreign exchange student over there.

子供がいません。
　I don't have children.

I like / I don't like

X は　Y が　adjective です。

X adjective Y.

Vocabulary

Kana Kanji Meaning
だいすき 大好き Love
すき 好き Like
きらい 嫌い Dislike
だいきらい 大嫌い Hate

Examples:

たけしさんはコーヒーが好きです。
Takeshi likes coffee.

ソラさんは納豆が大嫌いです。
Sora hates natto.

To be neutral, say:

好きでも嫌いでもないです。
I neither like nor dislike it.

To express familial or romantic affection, replace が with のことが.

たけしさんはメアリーさんが好きです。
Takeshi likes Mary (non-romatic)

たけしさんはメアリーさんのことが好きです。
Takeshi is in love with Mary.

To say "I like to do..."

Use short present form + の

Examples:

日本語を勉強するのが好きです。
I like studying Japanese.

部屋を掃除するのが嫌いです。
I don't like cleaning my room.

Please

～ください/～お願いします

Say the item that you want + ください OR お願いします
ください is more familiar, while お願いします is more formal.
Using を to indicate the direct object is optional.

Examples:

パンをください
Bread, please.

もっとください
More, please.

私の本お願いします。
Please give me my book.

～て下さい

て form + ください is used to politely request someone do something.

Examples:

教科書を読んで下さい。
Please read the textbook.

ちょうと待ってください。
Please wait a moment.

I Did This And That

て form is used to connect two verbs in a single sentence.

VERBて、VERB
This happened, then this happened.

Examples:

図書館に行って、本を借ります。
I will go to the library and borrow a book.

今日は、六時に起きて、勉強しました。
Today, I got up at 6 and studied.

バスに乗って、会社に行きます。
I go to work by bus.

と only connects nouns, so cannot be used here.

Quoted Speech

"I think that..."

To describe what you think, use short present + と思います。
To describe what you DON'T think, use short present + ないとおもいます。
To describe what you think happened in the past, use short past + と思います。

Examples:

たけしさんはメアリーさんが好きだと思います。
I think Takeshi likes Mary.

メアリさんはたけしさんが好きじゃないと思います。
I don't think Mary likes Takeshi.

たけしさんが豚カツを食べたと思います。
I think Takeshi ate my pork.

"They said..."

To quote what you heard someone say, use short present + と言っていました。
To quote what someone said, but you did not directly hear, use short present + といました。
To quote what someone said when the original speaker has used the past tense, use short past + と言っていました。

Examples:

ヤスミンさんは明日は試験があると言っていました。
Yasmin said there is a test tomorrow.

山下先生は結婚していないと言っていました。
Professor Yamashita said he is not married.

たけしさんは、昼ご飯を食べなかったと言っていました。
Takeshi said that he did not eat lunch.

When quoting speech, the original phrase needs to stay in the same tense as the speaker used. If you heard the speaker use present, use present. If you heard the speaker use past, use past.

Because

To explain a reason or cause:

(situation). (explanation) から。
(explanation) から、(situation)

Examples:

バスに乗りましょう。タクシーは高いですから。
Let's take the bus. Taxis are expensive.

明日テストがありますから、今晩勉強します。
Because there is a test tomorrow, I will study tonight.

Action in Progress

Use て form plus いる for:

-Activities in progress
-Results of a change

Examples:

ソラさんは英語を教えています。
Sora teaches English.

今、何をしていますか。
What are you doing right now?

山下先生は結婚しています。
Professor Yamashita is married.

家族は東京に住んでいます。
My family lives in Tokyo.

You cannot use ている with verbs that describe continuous states like ある, いる, etc.

I am going X to do Y.

If a person is going to a different location in order to do something:

Destination (に・へ) Verb Stem に (行く・来る・帰る)

Examples:

メアリーさんは日本に日本語を勉強しに来ました。
Mary came to Japan to study Japanese.

私はデパートにかばんをかいに行きました。
I went to the department store to buy a bag.

I am good/bad at...

Vocabulary

Kana Kanji Meaning Used For
じょうず 上手 Skilled Others
とくい 得意 Comfortable Self
にがて 苦手 Uncomfortable Others
へた 下手 Unskilled Self

To say "I am good/bad at doing..."

Short verb + の X です。

Make sure to use the polite term when describing skills.

Examples:

ロバートさんは料理を作るのが上手です。
Robert is good at cooking.

私は日本語を話すのが下手です。
I am bad at speaking Japanese.

何か・何も

Positive

To say "something" in a positive statement, use 何か

Examples:

猫が何かもってきました。
The cat brought something in.

Questions

To use "anything" in a question, use 何か

Examples:

猫は何か食べましたか。
Did the cat eat anything?

Negative

To say "not anything", use 何も + negative verb phrase

Examples:

いいえ、猫は何も食べませんでした。
No, the cat did not eat anything.

Not Yet/ Already

I have already...

To say you have already done something, use
もう + what you have done + ました

Examples:

もう宿題をしました。
I already did the homework.

もう朝ご飯を食べました。
I already ate breakfast.

I have not yet...

To say you have not yet done something, use
まだ + what you have not done + ていません

Examples:

まだ宿題をしていません。
I haven't done the homework yet.

まだ朝ご飯を食べていません。
I haven't had breakfast yet.
